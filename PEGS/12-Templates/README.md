# 12 — Templates

The friction-reduction layer: every recurring document type has a template
here, so drafting costs minutes and every document is born compliant with the
canon.

**Contains**

- [PEGS-DOC-TEMPLATE.md](PEGS-DOC-TEMPLATE.md) — any governance document:
  control block, constitutional anchor, numbered provisions, ratification record
- [DECISION-MEMO-TEMPLATE.md](DECISION-MEMO-TEMPLATE.md) — required for every
  Class 2 (one-way door) decision per PEGS-000 Art. V §1
- [SOP-TEMPLATE.md](SOP-TEMPLATE.md) — standard operating procedures for `09-SOP`

**Rules of this folder**

- Documents start from templates — a document type used twice earns a template
  (Art. III §5).
- Template changes ripple forward only; existing documents are not
  retro-edited to match.

**Custodian:** Founder until delegated.

bridge (the only new prose the playbook ever contains). The repo remains
the source of truth; the playbook is its readable projection
(INTEGRATION-STANDARD: one source, many projections).

## Governance notes

The playbook compiles; it never legislates. If a sentence exists only in
the playbook, it is either an editorial bridge (allowed, non-binding) or a
filing error (fix: move to canon, reference back).

## Implementation recommendations

   playbook PDF/site per merge ⚙) during Phase 7.
2. First human-bound edition printed for the first annual constitutional

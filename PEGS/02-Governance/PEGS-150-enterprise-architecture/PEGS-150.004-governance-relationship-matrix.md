## 4. Escalation procedures (uniform, all entities)

1. Role holder hits a Class 2 trigger → stop, write the decision memo →
   Founder (or chartered authority).
2. Firewall touch (clinical/legal/PHI, brand, compliance) → escalate
   immediately, no thresholds apply (PEGS-000 Art. VI §4).
3. Unclear class → treat as one class higher, escalate once (Art. V §1).
4. Cross-entity conflict → Holdings coordinates (TARGET); Founder until then.
5. Anything touching the Immutable Core → automatic fail (Art. VIII §2).

## Governance notes

No cell in this matrix grants authority by itself — authority flows from
ratified charters and matrices (L02); this document is the map of where
authority *lives*, not a source of it.

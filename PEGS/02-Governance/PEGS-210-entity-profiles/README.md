**Rules:** profiles update whenever an identity/ownership/governance fact
changes (same PR as the change) · confidential values (EIN, banking) stay
as placeholders — real values live with counsel/CFO · a new entity gets
its profile in the same act that charters it (PEGS-100 §4).

**Custodian:** Founder until delegated per entity.

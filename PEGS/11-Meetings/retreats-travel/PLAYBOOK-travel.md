# Travel Playbook

## Domestic trips (conferences, site visits, retreats)

**Before** — purpose in one line (what decision or relationship justifies
the seat) · budget + itinerary from the template · clinic/entity coverage
confirmed (the deputy path is real, not theoretical) · packet reading
loaded for transit time.

**During** — one daily 10-line log: who, what, what it means (feeds the
report; memory is not a system) · receipts to the expense folder same day ⚙.

**After (≤72h)** — trip report: outcomes vs. purpose, contacts worth
keeping (→ CRM/knowledge base), actions with owners/dates → tracker,
spend vs. budget.

## International strategy summits

The deep-work version: the Founder (± chosen leaders/advisors) leaves the
operating theater to think at decade scale.

| Element | Protocol |
|---|---|
| Justification | One decade-question named in advance; Class 2 memo if beyond budget threshold |
| Design | 3–5 days: mornings deep work on the question, afternoons input (reading, advisors, visits), evenings synthesis notes |
| Coverage | Full authority-matrix continuity: who holds what while the Founder is away — written before wheels-up |
| Security & compliance | Devices clean of PHI; VPN; no firewalled work over hotel networks |
| Output | Summit memo: the question, the thinking, the position reached, the decisions to memo formally on return |

**Rule for all travel:** if the purpose can't be stated in one line before
booking, the trip isn't ready to book.

# Meeting Type — Stewardship Council Annual Session (pre-activation)

> The Council is dormant by design (PEGS-101 §1) and holds no authority
> while the Founder is active. But dormant ≠ unrehearsed: once the
> sealed roster is named (06-Trust), stewards-designate convene once a
> year so that, if activation ever comes, it finds people who know the
> Constitution, the map, and each other. **Until the roster is named,
> this template waits — it does not convene.**

## Session parameters

| Parameter | Value |
|---|---|
| Cadence | Annually, adjoined to the Constitutional Review (July) |
| Duration | Half day |
| Convener | Founder |
| Attendees | Stewards-designate (sealed roster) + Founder; counsel optional for §4 |
| Authority exercised | **None** — this is formation, not governance (PEGS-101 §1) |
| Records | Kit minutes; filed to `06-Trust/family-records/` (sensitivity rules apply — never synced externally) |

## Standing agenda

| Block | Time | Content |
|---|---|---|
| 1. The Constitution, aloud | 60m | Joint participation in the Annual Constitutional Review (L01 SOP) — stewards read articles as rotating readers |
| 2. State of the map | 30m | PEGS-100 current entity map + PEGS-150.002 ecosystem walked end-to-end; changes since last year explained |
| 3. Succession readiness | 30m | Succession roadmap checklist status (06-Trust): instruments, deputies, incapacity process — deltas only, honestly |
| 4. **Activation drill** | 45m | Tabletop: "activation trigger fired this morning." Walk PEGS-101 §2–§5 — who convenes, what transfers, what the first 30 days look like, where the supermajority rules bite. Findings recorded |
| 5. Steward formation | 30m | Each steward-designate: one area of the canon studied this year + one question asked of the Founder while he can still answer it |
| 6. Closed reflection | 15m | Without the Founder present: does the Council believe it could serve? Gaps named in writing to the Founder |

## Preparation checklist

- [ ] Roster current (Founder reviews the sealed instrument annually)
- [ ] Packet T-14d: PEGS-000 full text · annual index · succession
      checklist status · prior session's findings
- [ ] Drill scenario chosen by the Founder (rotates: delegation /
      incapacity / death)
- [ ] Confidentiality reaffirmed in writing at each session

## Outputs

Kit minutes (Trust-filed) · drill findings → succession roadmap actions
· formation commitments per steward · the §6 gaps letter. These outputs
are PEGS-150.009's Level-4/5 gate evidence ("Council rehearsed").

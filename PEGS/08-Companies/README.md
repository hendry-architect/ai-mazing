# 08 — Companies

The operating layer: one subfolder per operating company, each holding that
company's charter, brand canon, and company-specific governance.

**Convention**

```
08-Companies/
  PassQual-Health/        ← charter, brand canon, provider governance
  <Next-Company>/         ← created only via the PEGS-100 §4 lifecycle
```

Each company subfolder contains, at minimum:

1. **Company charter** — mandate, class (per PEGS-100 §1), accountable chief
   executive, firewall obligations
2. **Brand canon** — voice, visual system, channel rules (the Brand Firewall
   of PEGS-000 Art. VI §4 applied to this company)
3. Pointers to its SOPs (`09-SOP`) and policies (`10-Policies`)

**Rules of this folder**

- A company without a charter here does not operate (PEGS-100 §4).
- The current authoritative list of companies is the entity map — PEGS-100 §2.
  This folder implements it; it does not define it.

**Status:** Active — company subfolders are created as charters are drafted
(PassQual Health first, in Phase 4).
**Custodian:** each company's chief executive; Founder until then.

# Nonprofit Governance Framework

Ready before the Foundation exists, so formation (a Class 2 decision) is a
checklist, not an invention. Legal formation with counsel; this frames the
governance substance.

## Formation gates

- [ ] Class 2 decision memo passed (Five Gates; Legacy gate weighs heaviest)
- [ ] Mission one-liner fits PEGS-000 Art. I (sustain, bilingual, health)
- [ ] Initial giving pillars chosen (≤3 — focus is the first governance act)
- [ ] Funding policy: seed + sustaining source named
- [ ] PEGS-000 adopted by the founding board on day one (PEGS-100 §5 logic)

## Board (when formed)

| Element | Standard |
|---|---|
| Size / composition | 3–7; majority independent within 3 years; one clinical voice while health programs run |
| Duties | Care, loyalty, obedience — taught at onboarding, not assumed |
| Conflicts | Advisor-agreement COI standard applies (`04-Advisory-Council`), plus IRS self-dealing rules |
| Meetings | Quarterly on the Meeting Kit; **board packet = Meeting Kit TEMPLATE-board-packet** with the impact dashboard as §2 |
| Founder's role | Founder chairs initially; succession per PEGS instruments |

## Program oversight

Each program: one owner · a logic model (who we serve → what we do → what
changes) · ≤5 outcome metrics · reviewed quarterly
([CHECKLIST-program-oversight.md](CHECKLIST-program-oversight.md)).

## Money discipline

Grants per [SOP-grant-approval.md](SOP-grant-approval.md) · no grants to
insiders · admin-cost band set and published · annual filings on the
governance calendar · impact reported annually
([TEMPLATE-impact-report.md](TEMPLATE-impact-report.md)) — in both
languages, like everything the family signs.

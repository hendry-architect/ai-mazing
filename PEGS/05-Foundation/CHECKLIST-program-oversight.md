# Checklist — Quarterly Program Oversight

> Run per program, per quarter, by the program owner; results roll into the
> board packet §2.

| # | Check | Y/N + note |
|---|---|---|
| 1 | Logic model still true? (who we serve → what we do → what changes) | |
| 2 | Outcome metrics current and moving? (≤5, each with actuals) | |
| 3 | Budget vs. actual within band? Variances explained? | |
| 4 | Grantee/partner reports received and reviewed? | |
| 5 | Any insider benefit, conflict, or compliance flag? (if yes → escalate now) | |
| 6 | Beneficiary voice heard this quarter? (one concrete input, not assumption) | |
| 7 | Consent documentation current for any story/photo used? | |
| 8 | What should the board decide or know? (one line, or "nothing") | |

**Sign-off:** program owner · date · filed to `programs/<name>/oversight/`.

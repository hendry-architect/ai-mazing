## 2. Programs

Per program: what we set out to change → what changed (measured) → what we
learned → what changes next year. One page each, no gloss: missed targets
reported with the same font size as hits (Integrity before advantage).

## 3. Stories (with written consent — always)

Two or three human stories that the numbers can't carry. Real names only
with documented permission; never patient/clinical details.

## 4. Money

Sources · uses · reserves. Simple enough for any family member to read
aloud and answer questions about.

## 5. Next year

Commitments in measurable terms — next year's report grades these.

*Filed to the board packet, the family meeting, and the public channel the
board designates.*

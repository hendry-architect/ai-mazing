# Checklist — Constitutional Alignment

> Run before any new initiative, entity, hire, campaign, acquisition, or
> automation goes live. Ten yes/no questions; any "no" stops the launch
> until resolved or escalated.

| # | Check | Y/N |
|---|---|---|
| 1 | Does this sustain the people it touches (Mission Gate)? | |
| 2 | Is every claim true and evidence-backed — no exaggerated certainty (Integrity Gate)? | |
| 3 | Would we sign this work as-is (Standard Gate)? | |
| 4 | Does it compound into a system, asset, or capability (Leverage Gate)? | |
| 5 | Will it look like a good decision in ten years (Legacy Gate)? | |
| 6 | Decision class identified (1/2/3) — and if Class 2, is the decision memo written? | |
| 7 | Clinical/Legal/PHI firewall respected — nothing delegated below licensed authority? | |
| 8 | Brand firewall respected — personal brand not selling; commercial brand not borrowing the Founder's voice? | |
| 9 | Compliance floor met (HIPAA, FTC, licensure, applicable law)? | |
| 10 | Custodian/owner named, and the decision recorded in writing? | |

**Sign-off:** name · role · date · initiative · result (proceed / revise /
escalate).
